API
API is Application Programming Interface, that allows two applications to talk each othrer.
Each time you use app like, Send instant message, or check the wather on your phone, you are using APU

What is Restful API?
Rest stand for Representational State Transfer
Rest sometimes known, RESTful APIs.
REST or RESTful APIs were sedigned to take advantage of existing protocols

Client-Server
This constraint operates on the concept that the client and the server should be separate from each other and allowed to envolve individually

Stateless
REST APIs are stateless, meaning that calls can be made independently of one another, and each call contains all of the data necessary to complete itself successfully

Use HTTP methods
REST API should use proper http methods:
GET, POST, PUT, DELETE

Send data in JSON (Mostly)
The API should send data back in form of JSON(Javascript Object Notation)
